## What is Orium?

Orium is a unified, open-source AI infrastructure — combining **Agent Orchestration**, **Universal Protocol Gateway**, **Application Development Platform**, and **AI-Native Runtime** into a single, cohesive system.

Built for **every device**, **every user**, **every model**.

## Quick Start

```bash
# One-line install
curl -fsSL https://orium.dev/install | sh

# Or clone
git clone https://github.com/your-org/orium.git
cd orium && npm install && npm run dev
```
