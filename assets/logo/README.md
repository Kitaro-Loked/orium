# Orium Logo Design

## Design Philosophy

### Name Origin
**Orium** = "Or" (Latin for gold/precious metal) + "ium" (chemical element suffix)

The logo embodies three core concepts:
1. **Element/Crystal** - The hexagonal core represents atomic structure and the "element" concept
2. **Orbit/Network** - Concentric orbital rings represent AI model connectivity and universal infrastructure
3. **Intelligence Core** - The glowing center represents the AI brain at the heart of the system

## Color System

| Token | Hex | Usage |
|-------|-----|-------|
| Deep Sea | `#0A2540` | Primary background, depth |
| Indigo | `#6366F1` | Primary accent, orbit rings |
| Light Indigo | `#818CF8` | Nodes, highlights |
| Pale Indigo | `#A5B4FC` | Secondary nodes |
| White | `#FFFFFF` | Core glow, text |

## Variants

### 1. Full Logo (`logo.svg`)
200×200px, complete with all orbital details. Use for app icons, splash screens, and brand materials.

### 2. Mark (`logo-mark.svg`)  
64×64px, simplified version. Use for favicon, small UI elements, and compact spaces.

### 3. Wordmark (`logo-wordmark.svg`)
400×80px, logo + text. Use for headers, documentation, and marketing materials.

## Usage Guidelines

- **Minimum size**: Mark should not be smaller than 24×24px
- **Clear space**: Maintain padding equal to the core hexagon width around the logo
- **Dark backgrounds**: Use the full color version on dark backgrounds
- **Light backgrounds**: The wordmark works best; the mark can be used with the gradient
